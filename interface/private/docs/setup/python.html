Python Setup Guide

Installing Python

This system requires a minimum python version of 3.12. You can find the lastest download from here.
Once python is installed, in the project directory, run the command python -m venv .venv. This will create our python virtual environment.
Next, we need to tell our computer to use the resources Python has made for us.

Use the table below to find what command to run:
System Command
Windows Powershell .\.venv\Scripts\Activate.ps1
FISH Shell source .venv/bin/activate.fish
ZSH Shell source .venv/bin/activate
C-Shell source .venv/bin/activate.csh
Bash Shell source .venv/bin/activate

Now, we need to install all our Python Packages for this program to run. Run the command pip install -r requirements.txt.
This will install all the requirements that the program requires to run correctly.

Congratulations! Python is now configured for running this project!
