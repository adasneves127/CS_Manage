Finance Types

Financial Record Types are the manner in which this system organizes types of Finances.
The following types are given by the system on creation:
  • Budget — Used for records requesting money from the Student Government Association. While this item is debated, it should be approved by a System Account which represents the Student Government Finance Committee. Totals should be a negative ammount to reflect a negative outgoing ammount, since the money is comming into the account.
  • Invoice — Used for records paying money to other organizations. Totals should be a positive number, to reflect money going out of the account.
  • Credit
This system allows Finance Administrators to modify the types available to users in the system. To do so, use the navbar on the application pages, and go to Admin -> Record Types.
This page allows you edit existing Finance Types as well as create new items. Please note that editing existing types is not reccomended, as updating an existing type will update all records which use that type already.
